# telegram-session-generator
Generate Telegram string session

### usage
```bash
git clone git@github.com:hirenreshamwala/telegram-session-generator.git
cd telegram-session-generator
npm install
cd telegram-app
npm install && npm run build
cd ..
node server.js
```
